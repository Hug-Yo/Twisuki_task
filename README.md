一个项目应该有个 `README.md` 文档, 解释下这是什么, 怎么用, 你都干了什么.

虽然我没有创建 `git`, 但是它仍然重要.

`README.md` 没有固定的格式, 但至少应该包括 `## 安装和启动`, `## 使用方法(或功能介绍)`.

请先阅读 `README.md`, 之后阅读 `main.py`, `data/README.md`.

之后按顺序阅读/攻关 `__init__.py`, `main.py`, `model.py`, `crud.py`, `filter.py`.

之后比较阅读 `command.py` 和 `controller.py`.

最后再回到 `main.py`, 完成编写.
